Permissions project â€” brief

0-iam_betty: switch to user betty (command: "su betty")
1-who_am_i: print effective username (whoami)
2-groups: print groups for current user (groups)
3-new_owner: change owner of file "hello" to betty (chown betty hello)
4-empty: create an empty file called "hello" (touch hello)
5-execute: add execute permission to owner of "hello" (chmod u+x hello)
6-multiple_permissions: add execute to owner & group, read to others (chmod ug+x,o+r hello)
7-everybody: add execution permission to owner, group and others (chmod a+x hello)
8-James_Bond: set owner & group no permissions, others all (chmod 007 hello)
9-John_Doe: set mode to -rwxr-x-wx (chmod 753 hello)
10-mirror_permissions: set "hello" mode same as "olleh" (chmod --reference=olleh hello)
11-directories_permissions: add execute permission to all subdirectories (find . -type d -print0 | xargs -0 chmod a+x)
12-directory_permission: create directory my_dir with permissions 751 (mkdir -m 751 my_dir)
13-change_group: change group owner to school for "hello" (chgrp school hello)
14-change_owner_and_group: change owner to vincent and group to staff for all files/dirs (chown vincent:staff *)
15-symbolic_link_permissions: change owner/group of symbolic link _hello (chown -h vincent:staff _hello)
16-if_only: change owner to vincent only if currently owned by guillaume (chown --from=guillaume vincent hello)
